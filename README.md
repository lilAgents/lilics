# lilICS

Add to Calendar Link Generator

Turn event details into add-to-calendar links for Google, Outlook, and Yahoo, plus a downloadable .ics file and a paste-ready snippet.

Part of [Hot App Summer](https://hotappsummer.netlify.app) by [lilAgents](https://lilagents.com): one small, useful web tool shipped every day through June.

## Run it locally

This is an [Astro](https://astro.build) site.

```bash
pnpm install
pnpm dev
```

To build for production:

```bash
pnpm build
```
